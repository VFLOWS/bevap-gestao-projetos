────────────────────────────────────────────────────────────────────────────────

  • Primary Green: #1C8C5D (ações principais, elementos importantes)

Tipografia:

  • Inter: Corpo de texto (weights: 300-700)

📦 WIDGETS IMPLEMENTADOS
────────────────────────────────────────────────────────────────────────────────

1. bevap-shared (Recursos Compartilhados)
   └─ bevap-colors.css: Estilos e variáveis CSS
   └─ bevap-utils.js: Biblioteca de utilitários ES5
      • showLoading(), showEmpty(), showError()
      • formatDate(), formatDateTime()
      • openDrawer(), closeDrawer()
      • escapeHtml() - Proteção XSS
      • ajax() - Requisições HTTP

2. bevap-dashboard (Dashboard Principal)
   └─ KPIs animados com contadores
   └─ Gráfico de distribuição por status
   └─ Tabela de projetos recentes
   └─ Métricas: Total, Em Desenvolvimento, Validação, Concluídos

3. bevap-solicitacao (Gestão de Solicitações)
   └─ Formulário em drawer lateral
   └─ Tabela com filtros (status, prioridade, busca)
   └─ Estados: Novo, Em Análise, Aprovado, Rejeitado
   └─ Campos: Nome, Descrição, Prioridade, Área, Prazo, Justificativa

4. bevap-planejamento (Kanban)
   └─ 5 colunas: Backlog → A Fazer → Em Análise → Aprovado → Pronto
   └─ Drag & Drop entre colunas
   └─ Contadores dinâmicos
   └─ Badges de prioridade

5. bevap-desenvolvimento (Tracking)
   └─ Cards de projetos
   └─ Barra de progresso visual (cores dinâmicas)
   └─ Métricas: Sprint, Tarefas (concluídas/total), Datas
   └─ Progresso: Verde (80%+), Amarelo (50-79%), Azul (30-49%), Vermelho (<30%)

6. bevap-backlog (Gestão de Backlog)
   └─ Tabela de itens
   └─ Filtros: Prioridade, Tipo
   └─ Tipos: Feature, Bug, Melhoria
   └─ Estimativas em pontos
   └─ Ação: Mover para planejamento

7. bevap-atividade (Timeline)
   └─ Timeline vertical cronológica
   └─ Tipos: Criação, Atualização, Comentário, Tarefa
   └─ Ícones coloridos por tipo
   └─ Data/hora e usuário responsável

8. bevap-validacoes (Validações)
   └─ Duas abas: Solicitante + TI
   └─ Estados: Pendente, Aprovado, Rejeitado
   └─ Ações: Aprovar / Rejeitar com feedback
   └─ Campo para justificativa

9. bevap-entrega (Delivery)
   └─ Go-Live: Checklist interativo com progresso
   └─ Treinamentos: Gestão de capacitação
   └─ Encerramento: Lições aprendidas + documentação final

10. bevap-templates (Biblioteca)
    └─ Grid responsivo de templates
    └─ Filtros: Workflows, Formulários, Dashboards
    └─ Ações: Visualizar / Usar template
    └─ Ícones específicos por categoria

✨ CARACTERÍSTICAS TÉCNICAS
────────────────────────────────────────────────────────────────────────────────

Frontend:
  • Tailwind CSS 3.x via CDN (sem build necessário)
  • JavaScript ES5 (compatibilidade máxima, IE11+)
  • HTML/FTL templates semânticos
  • Google Fonts (Montserrat + Inter)
  • Responsive design (mobile-first)

Acessibilidade (WCAG AA):
  • ARIA labels e roles em todos os elementos
  • ARIA live regions para conteúdo dinâmico
  • Navegação por teclado completa
  • Focus visível customizado
  • Suporte a leitores de tela
  • Contraste adequado de cores
  • Touch-friendly (≥44px)

Segurança:
  • Escape de HTML (XSS prevention)
  • Sanitização de inputs
  • Validação client-side
  • Recomendações para validação server-side

Performance:
  • Carregamento via CDN
  • Estados de loading otimizados
  • Sem dependências pesadas
  • Código modular e reutilizável
